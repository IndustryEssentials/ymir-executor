# ymir1.3.0 (2022-09-30)

- 支持分开输出模型权重，用户可以采用epoch10.pth进行推理，也可以选择epoch20.pth进行推理

- 训练镜像需要指定数据集标注格式, ymir1.1.0默认标注格式为`ark:raw`

- 训练镜像可以获得系统的ymir版本，方便镜像做兼容
